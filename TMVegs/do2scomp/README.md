# Minimal example for compilation

There are two examples here, with minimal difference.  The example compiled
output is in the two `.out` files.

To compile:
- For the 'standard' version: `tmc 2sc.yml main.yml shift.yml`
- For the alt version: `tmc 2sc.yml main_alt.yml shift.yml -o a_alt.out`

Note that the order of arguments matters only in so much that the output
filename must be directly after the `-o`.
